## Project Summary

Mod Menu is set up as an online catalog centered on War Robots mod menu content. It is meant for users who prefer a single web-accessible location to review, manage, and retrieve connected resources instead of searching across multiple scattered sources.

The repository uses a script-collection style structure so the available materials stay grouped and easy to reach from one place. The web build is the main delivery point for updates, with the goal of preserving a clean library layout and keeping the repository in step with the current content set.
